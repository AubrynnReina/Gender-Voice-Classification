# Gender-Voice-Classification

This is a project using a combination of machine learning algorithms and neural network to classify the biological gender of a human voice in Vietnamese. The input can be of any length, but should only be recorded with one person.

The dataset used is the Vietnamese Common Voice dataset. Remember to choose the corpus, not the delta segment. Here's the original dataset: https://commonvoice.mozilla.org/en/datasets
